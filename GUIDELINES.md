 # Wiki Contribution Guidelines

## 1. Naming Convention
All pages must follow this structure: Namespace:Descriptor (e.g., Hardware:Toei_TC-L175).

## 2. Namespaces
- Hardware: For cabinet components, CRT chassis, and mechanical parts.
- Mechanic: For gameplay systems, breeding loops, and race logic.
- Event: For tournaments like the Breeder's Cup.
- Logistics: For sourcing, transport, and acquisition guides.

## 3. Templates
All new pages must use the relevant Infobox template. Check the /templates directory in the Git repo.
